# kpim
**Personal Information Manager in cmd line**

Written in python3 using a sqlite db for storing data.

--

I create these scripts for manage few information I used often.

Main script is **kpim.py**, you need this one for launching other.

- kpim-contacts.py: manage your contact in command line
- kpim-css.py: manage the cheatsheet in command line
- kpim-link.py: manage url favorite in command line
